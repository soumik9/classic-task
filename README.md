## Frontend Features

- [x] Home Page with products
- [x] Products Page
- [x] Signup page
- [x] Signin page
- [x] Logout (after authentication)
- [x] Add to cart
- [x] Dashboard to show cart items

- [x] Jotai

- [x] React hot toast & Sweetalert
